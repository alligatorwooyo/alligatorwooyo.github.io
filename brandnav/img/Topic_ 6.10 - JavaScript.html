6.10 - JavaScript

Discussion Topic: 6.10 - JavaScript6.10 - JavaScript

JavaScript is a scripting language that adds interactivity and flexibility to a web page. While you don't need to be a JavaScript expert to create functional web designs, understanding the connection between a web page and JavaScript can enhance a page's functionality.

These two videos introduce the use of JavaScript to create a functional mobile menu. Although the focus of this semester is on an introduction to web design through HTML and CSS, JavaScript can change the layout of a web page and add functionality to it.

The video introduces the id selector in JavaScript, which can locate elements on a web page based on their position in the HTML document, class, id, query selector, or tag. CSS also uses these selectors.

Definitions:

  • Class: An HTML attribute used to define a class name for an element, which can be used to apply styles or JavaScript behaviors to multiple elements that share the same class name.
  • ID: An HTML attribute used to uniquely identify an element within a document. It is used by JavaScript to manipulate specific elements on the page and can also be used to create anchor links within a page.
  • Query Selector: A method in JavaScript used to select and manipulate elements in a document using CSS-like syntax. It allows developers to target specific elements based on their class, ID, attributes, or position within the document hierarchy.
  • Tag: A basic building block of HTML, used to define different types of content on a web page, such as headings, paragraphs, links, images, and more. Tags are defined by enclosing them in angle brackets, with the tag name inside.




JavaScript has the capability to manipulate the Document Object Model (DOM), which is the structure that represents the elements and content of a web page.

Through JavaScript, web designers and developers can dynamically update and change the content of a web page or of an interactive browser experience, as well as add or remove elements from the page.

Additionally, with the use of frameworks such as React JavaScript can create its own virtual DOM, which allows for more efficient updates and rendering of web pages. This ability to manipulate and create the DOM is one of the key features of JavaScript and is what enables the creation of interactive and dynamic web pages.

While the use of the DOM is a more advanced topic beyond the scope of the class, knowing of its existence will be helpful in the future. Why is React used? For the quickness in changing the appearance of a webpage via the DOM. Understanding the benefits of a language or library may in the future open up your thinking of creative uses for your art.
